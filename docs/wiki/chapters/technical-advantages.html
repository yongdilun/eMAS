Overview

The Factory Agent is not just another wrapper around an LLM. It is a comprehensive safety-first framework designed for high-stakes manufacturing environments. By combining state-of-the-art AI tooling with rigid software engineering patterns, we deliver a system that is both intelligent and industrial-grade.

🛠️ Advanced AI Tooling

Promptfoo: Matrix-Based Testing

We use Promptfoo to run matrix tests across hundreds of test cases. This ensures that changes to our reasoning prompts don't introduce logic regressions or safety violations.

LangSmith: Production Observability

Native integration with LangSmith allows us to trace multi-step agent loops, analyze token costs, and debug latency in real-time. This provides the transparency required for industrial audit trails.

RAG Retrieval: Grounded Safety

Uses ChromaDB and hybrid search to retrieve safety documents and SOPs. The agent grounded in actual factory rules, not just general LLM knowledge.

🛡️ Industrial Safety Features

1. Transactional Idempotency

The Feature: The agent calculates unique Idempotency Keys for every action.

The Value: Prevents "double-start" or "double-stop" errors on heavy machinery during network retries. Ensures exactly-once execution.

2. Cross-Step Reference Resolving ($ref)

The Feature: Support for symbolic references between tools in a single execution bundle.

The Value: Enables atomic planning of complex sequences (e.g., creating a job and assigning it to a machine) before any action is physically executed.

3. Human-in-the-Loop (HITL) Interrupts

The Feature: Native LangGraph breakpoints for high-risk write operations.

The Value: Acts as a "Safety Brake," forcing human verification before the agent commits to any physical side-effect.

4. Self-Healing Recovery Sweep

The Feature: Cold-start reconciliation logic that handles session inconsistencies after server restarts.

The Value: Detects and heals "stuck" sessions, preventing industrial state leakage and ensuring task continuity.

5. Intent Decomposition (Phase 3)

The Feature: A recursive splitter that breaks compound queries into dependency-aware tasks.

The Value: Handles complex orchestration logic (e.g., cascades) that typically causes simpler agent architectures to fail.
