0. Notes
--------

- Node benchmarks are opt-in. They run only when FACTORY_AGENT_RUN_NODE_BENCHMARKS=1.
- Some full Python runs on this Windows machine may need TMP/TEMP set to a repo-local folder.
- Live LLM/RAG commands are intentionally opt-in and depend on local/provider env.
- Seeded, real LangGraph, release, synthetic, reliability, security, privacy, and operational Playwright lanes are not the default PR gate unless the team promotes them.
